## Usage

1- Install the dependencies.

```sh
npm install
```

2- run the seeding script to feed the database with 100 tours documents.

```sh
npm run seed
```

3- transpile the components.

```sh
npm run build
```

4- then run the server.

```sh
npm start
```

5- go to `http://localhost:3003` 

## Requirements

- Node 8.11.4
- npm 5.6.0
- MongoDB
